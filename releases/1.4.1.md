BCRHP 1.4.1 release notes

Minor bug fix release

### Bug fixes and enhancements
- Fix dependabot critical alerts [#246](https://github.com/bcgov/bcrhp/issues/246)
- Allow dates before 1800 [#254](https://github.com/bcgov/bcrhp/issues/254)
- Remove update and legacy workflow cards [#257](https://github.com/bcgov/bcrhp/issues/257)
- Fix tooltips in Site Images page
- Updates to 1.4.0 readme

### Dependency changes:
None

### 1. Perform Application deployment
The source code branches are as follows in the `cd/docker/config/app_env` file:
```yaml
FORCE_CORE_PIP_INSTALL=True

CORE_REPO_TAG=v8.0.10-bcgov
COMMON_REPO_TAG=v2.0.1
CONFIG_BRANCH=v1.4.1
```
